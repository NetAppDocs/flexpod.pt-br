= Plataformas FlexPod
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./../media/


[role="lead"]
Existem duas plataformas FlexPod:

* *FlexPod Datacenter.* Esta plataforma é uma infraestrutura de data center virtual massivamente dimensionável adequada para aplicações empresariais de carga de trabalho, virtualização, VDI e nuvem pública e privada. FlexPod Datacenter tem suas próprias especificações, que estão documentadas em https://docs.netapp.com/us-en/flexpod/fp-def/dc-tech-spec_solution_overview.html["Especificações técnicas do data center FlexPod"^].
* *FlexPod Express.* Essa plataforma é uma infraestrutura convergente compacta destinada a casos de uso de borda e escritório remoto.


Este documento fornece as especificações técnicas da plataforma FlexPod Express.
