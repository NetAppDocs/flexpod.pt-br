== Requisitos de software para FlexPod Express com NetApp AFF/ ASA/ FAS

Os requisitos de software para o FlexPod Express com NetApp AFF/ ASA/ FAS incluem:

* ONTAP 9.16 ou posterior
* Cisco NX-OS versão 10.4(6)M ou posterior



NOTE: Todas as versões de software listadas nesta seção são os requisitos mínimos para os componentes de hardware (NetApp e Cisco) listados nas seções subsequentes deste documento.

Todos os softwares devem ser listados e suportados no http://support.netapp.com/matrix/["NetApp IMT"^]. Alguns recursos de software podem exigir versões mais recentes de código do que os mínimos listados em arquiteturas anteriores.
