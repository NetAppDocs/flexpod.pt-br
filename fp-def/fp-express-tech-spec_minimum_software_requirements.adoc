== Requisitos de software para FlexPod Express com NetApp AFF ou FAS

Os requisitos de software para o FlexPod Express com NetApp FAS incluem:

* ONTAP 9.1 ou posterior
* Cisco NX-OS versão 7.0(3)I6(1) ou posterior
* Na configuração gerenciada do Cisco UCS, Cisco UCS Manager UCS 4.0(1b)


Todo o software deve estar listado e ser compatível na  http://mysupport.netapp.com/matrix["NetApp IMT"^]. Certos recursos de software podem exigir versões de código mais recentes do que os mínimos listados em arquiteturas anteriores.
