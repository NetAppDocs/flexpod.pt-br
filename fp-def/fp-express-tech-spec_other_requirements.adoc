= Outros requisitos
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./../media/


[role="lead"]
Os requisitos adicionais para o FlexPod Express incluem o seguinte:

* Se você planeja usar o VMware ESXi como hipervisor, consulte a link:https://compatibilityguide.broadcom.com/["Guia de compatibilidade da Broadcom"] para verificar se seus componentes de hardware e software são compatíveis.
* Para uma implantação baseada em OpenShift, certifique-se de que você possui os componentes de hardware e software necessários que são compatíveis com OpenShift. Consulte o link:https://catalog.redhat.com/en/search["Catálogo do OpenShift"] para mais informações.
