NOTE: A inicialização SAN é opcional, mas altamente recomendada, pois permite a movimentação do perfil do servidor durante upgrades ou substituições de hardware.



== Opção de inicialização iSCSI

Os requisitos mínimos para a opção de inicialização iSCSI incluem:

* Uma licença/funcionalidade iSCSI ativada no controlador de armazenamento NetApp
* Um adaptador Ethernet 10Gbps de duas portas em cada nó no par de HA do controlador de storage NetApp
* Um adaptador no servidor Cisco UCS que é capaz de inicializar iSCSI




== Opção Cisco UCS X-Series Direct e AFF/ASA/FAS de conexão direta

NetApp storage controllers podem ser conectados diretamente aos interconectores de malha do Cisco UCS sem qualquer switch SAN upstream. FlexPod Express oferece suporte tanto à inicialização por FC quanto por iSCSI com o UCS X-Series Direct.

Os requisitos de licenciamento e hardware são os seguintes:

* É necessária uma licença de protocolo no controlador de storage NetApp.
* Um adaptador Cisco UCS (iniciador) é necessário no servidor. Para obter uma lista de adaptadores Cisco UCS suportados, consulte o NetApp http://mysupport.netapp.com/matrix["IMT"^] .
* É necessário um adaptador de destino no controlador de armazenamento NetApp.


A figura a seguir mostra uma topologia UCS X-Series Direct e ASA conectada diretamente para SAN baseada em IP.

image:fp-express-tech-spec_image10.png["Topologia Direct e direct-attached ASA da UCS X-Series para SAN baseada em IP"]

*Notas:*

* O Cisco UCS está configurado no modo de comutação Ethernet.
* As interconexões de portas iSCSI do destino para a malha são configuradas como portas de storage Ethernet para dados iSCSI.


A figura a seguir mostra uma topologia UCS X-Series Direct e ASA conectada diretamente para SAN baseada em IP e baseada em FC.

image:fp-express-tech-spec_image11.png["Topologia direta e de conexão direta ASA da UCS X-Series para SAN baseada em IP e FC"]

*Notas:*

* O Cisco UCS é configurado no modo de comutação FC.
* As portas FC do destino para a malha, as interconexões são configuradas como portas de storage FC.


Para obter mais informações sobre esta configuração, consulte  https://www.netapp.com/pdf.html?item=/media/135828-flexpod-san-ucs-xdirect-asa-nva-deploy-guide.pdf["FlexPod Solução SAN com Cisco UCS X-Series Direct e NetApp ASA"^].


NOTE: Os exemplos acima utilizam o controlador de armazenamento NetApp ASA, mas a mesma configuração é compatível com os controladores de armazenamento NetApp AFF e FAS.
