[[cisco-ucs-x-series-direct]]
== Cisco UCS X-Series Direct

Cisco UCS X-Series Direct é uma nova arquitetura que integra um par de Cisco UCS Fabric Interconnect 9108 100G com Cisco UCS X9508 Chassis e Cisco UCS X-Series Compute nodes para fornecer a solução UCS X-Series Direct.

Para uma visão geral rápida dos recursos e funcionalidades da série Cisco UCS X, consulte link:https://www.cisco.com/c/en/us/products/collateral/servers-unified-computing/ucs-x-series-modular-system/ucs-x-series-direct-aag.html["Visão geral do Cisco UCS X-Series Direct"^].

Cisco UCS X-Series Direct é composto pelos seguintes componentes:

* Cisco UCS 9108 100G Fabric Interconnect
* Chassis Cisco UCS X9508
* Cisco UCS X-Series Nós de computação
* Componentes opcionais:
+
** Nós PCIe e módulo X-Fabric
** Segundo chassi Cisco UCS X9508 com nós de computação da série X, módulos de malha inteligente Cisco UCS 9108-100G e até quatro servidores de rack da série C





NOTE: Consulte este link para obter a especificação técnica completa do UCS X-Series Direct: https://www.cisco.com/c/en/us/products/collateral/servers-unified-computing/ucs-x-series-modular-system/ucs-x-series-direct-ds.html["Cisco UCS X-Series Folha de dados direta"].

[[cisco-unified-edge]]
== Cisco Unified Edge

Cisco Unified Edge é um sistema preparado para IA que redefine computação na borda, convergindo computação, rede, armazenamento e segurança. O design modular é pronto para o futuro, eficiente em termos de energia e fácil de manter, podendo ser personalizado para suportar as cargas de trabalho e casos de uso atuais, mantendo-se adaptável ao cenário de IA em rápida evolução.

Para uma visão geral rápida dos recursos e funcionalidades do Cisco Unified Edge, consulte link:https://www.cisco.com/c/en/us/products/collateral/servers-unified-computing/unified-edge/unified-edge-at-a-glance.html["Cisco Unified Edge Visão geral."^]

Cisco Unified Edge consiste nos seguintes componentes:

* Chassis Cisco UCS XE9305
* Dois Cisco Unified Edge Management Controllers com switches 25G incorporados
* Nós de computação: escolha de
+
** Nó de computação Cisco UCS XE130c M8
** Nó de computação Cisco UCS XE150c M8
