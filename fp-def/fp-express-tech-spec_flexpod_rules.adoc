= Regras do FlexPod
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./../media/


[role="lead"]
O projeto FlexPod suporta uma arquitetura de infraestrutura flexível que abrange uma variedade de componentes de hardware e versões de software.

Utilize os conjuntos de regras como guia para construir ou montar uma configuração válida de FlexPod. Os números e regras listados neste documento representam os requisitos mínimos para FlexPod e podem ser expandidos utilizando produtos compatíveis dentro das famílias de produtos incluídas, conforme necessário para diferentes ambientes e casos de uso.
